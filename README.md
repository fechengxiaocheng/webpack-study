## 开始使用

```
// 下载
git clone https://github.com/fechengxiaocheng/webpack-study
// 进入到某demo文件夹
cd my-webpack-app-0
// 安装
npm install
// 部署
npm run build
// 本地运行(如果package.json中定义了的话)
npm start 或 npm run server
```
